# EAAO factory governance (backup)

These are the original repo-root governance files of the **EAAO factory** as they stood
before this repository was turned into the `d4np-c` library. They govern *work on the
factory itself* (see AGENTS.md §1). The repo root now carries the **library's** governance
(which governs d4np-c); these copies are kept so the factory contract is not lost.

To work on the factory, read these; to work on the library, read the root `AGENTS.md`.
