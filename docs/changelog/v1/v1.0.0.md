# v1.0.0 — 2026-06-21

First stable release of `d4np-c`. All 25 specified functions are implemented across the seven
modules, verified under the full quality bar (build matrix, ASan/UBSan/TSan, Valgrind, ≥80% line
coverage, warning-free Doxygen, broad-reach strict-C11 build). The public ABI is frozen for the
1.x line (see ADR-0008).

## Added

- Project bootstrap: enterprise scaffolding (agent contract, docs system, CI gate, consistency
  lint).
- `d4np/core`: `d4np_status_t` status codes + `d4np_status_str`; pluggable `d4np_allocator_t`
  with a default libc-backed allocator; `version.h`.
- `d4np/str`: non-owning `d4np_str_view_t` with zero-allocation `d4np_str_view_split_next` (#15);
  `d4np_str_split` into a caller-provided view array (#16); `d4np_str_parse_int` (base 2..36 or
  auto, overflow-safe, full int64 range) and `d4np_str_parse_float` (#17), returning
  `d4np_status_t`.
- `d4np/mem`: arena allocator `d4np_arena_init/alloc/reset/destroy` (#1-#3) and fixed-block pool
  `d4np_pool_init/alloc/free/destroy` (#4-#5), both O(1); benchmarks under `bench/mem`.
- `d4np/ds`: generic `d4np_vector_t` (#6); `d4np_string_builder_t` (#10); `d4np_hashmap_t` (open
  addressing + linear probing, pluggable hash/eq, default FNV-1a) (#7); intrusive
  `d4np_linked_list_t` (#8); lock-free SPSC `d4np_ring_buffer_t` over C11 atomics (#9, TSan-tested);
  benchmarks under `bench/ds`.
- `d4np/concurrency`: portable `d4np_mutex_t` (#11); in-process counting `d4np_semaphore_t` (#14)
  and the named, cross-process `d4np_named_semaphore_t` (#14, `sem_open` / named `CreateSemaphore`,
  multi-process verified); lock-free unbounded SPSC `d4np_atomic_queue_t` (#13); `d4np_thread_pool_t`
  with graceful drain (#12); benchmarks under `bench/concurrency`.
- `d4np/io`: `d4np_file_read_all` (#18); `d4np_file_write_all` (durable atomic write) (#19);
  `d4np_path_combine` (#20).
- `d4np/sys`: `d4np_hash_fnv1a` (+ str/continuation) (#25); `d4np_timestamp_ms`/`_ns` (monotonic)
  (#23); `d4np_uuid_generate` + `d4np_uuid_format` (RFC4122 v4) (#24); thread-local
  `d4np_error_context_*` (#22); leveled `d4np_log_write` (#21).
- Build/CI/docs: ≥80% line-coverage gate via gcovr (ADR-0003); warning-free Doxygen API reference
  + per-module README quickstart (ADR-0004); CMake install/export with a vcpkg port and Conan
  recipe (ADR-0005); strict-C11 broad-reach conformance gate + compatibility matrix (ADR-0006);
  multi-process IPC harness and a performance gate (ADR-0007).
